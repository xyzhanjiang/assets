# slick

[slick](https://github.com/kenwheeler/slick)

Installation via npm: `npm install slick-carousel --save`

example:

``` html
<div data-slick='{"slidesToShow": 4, "slidesToScroll": 4}'>
  <div><h3>1</h3></div>
  <div><h3>2</h3></div>
  <div><h3>3</h3></div>
  <div><h3>4</h3></div>
  <div><h3>5</h3></div>
  <div><h3>6</h3></div>
</div>
```

需要手动调用插件：

``` javascript
$('[data-slick]').slick()
```
